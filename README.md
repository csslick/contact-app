# contact-app 연락처 앱

## 소개
1. 연락처 추가 및 조회
2. 연락처 삭제(아이콘 클릭)
3. 전화걸기
4. 연락처 저장

## 구현 기술(JAVASCRIPT)
- 배열 조작: 추가, 삭제
- 로컬 스토리지 읽고 쓰기, JSON parsing
- 동적 추가 요소 이벤트 처리

## 활용 라이브러리
- 제이쿼리
- Bootstrap Icons
